Funciones

funciones es una estructura del siguiente tipo, pueden madarse parametros o no
funcion nombreFuncion(){
bloque de codigo o sentencias, no podra estar vacio
}
tambien se presenta una funcion principal su estructura es
funcion principal{
bloque de codigo o sentencias, no podra estar vacio
}

Ciclo

Se acepta la sentencia mientras, la cual contiene una condicion para seguirse repitiendo el ciclo hasta no cumplir con la sentencia, la estructura es la siguiente
mientras(booleano){
bloque de codigo no podra venir vacio
}

Escritura

Condicion para poder escribir en pantalla cadenas de texto, variables como enteros, identificadores, flotantes la estructura es la siguiente
imprimir();
dentro de los parentesis se encuentra lo que se desea escribir

Variables

las variables son declaradas e iniciadas con cadenas, enteros, identificadores, flotantes desde el momento que se declaran la estructura es la siguiente
variable nombreVariable = ;
despues del igual se coloca que se va a inicializar
tambien se pueden realizar otras asignaciones siendo enteros, decimales, cadenas o otras variables e identificadores despues de ser declaradas su estructura sera la siguiente
nombreVariable = ;

Condiciones

Son funciones que dentro de el llevan sentencias que devuelven verdadero o falso, en este caso son booleanos, si se cumple entra al bloque de codigo sino entra a otro bloque de codigo
si(booleano){
bloque de codigo
}sino{
bloque de codigo
}

Bloque de codigo

Dentro del bloque que codigo pueden venir ciclos, llamadas a funcion,impresiones en pantalla, declaracion de variables, asignacion de variables, condiciones todo lo mencionado anterior puede venir, acepcion que dentro de una funcion no podra declararse otra funcion.
